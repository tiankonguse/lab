  • int vline(ch, n)  
    int wvline(win, ch, n)  
    int hline(ch, n)  
    int whline(win, ch, n)  
    These functions draw a vertical or horizontal line starting at the current cursor position. ch is the character to use and n is the number of characters to draw. The cursor position is not advanced.
